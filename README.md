UFCpy is a Python wrapper to get access to the UFC fighter roster. 

## Installation

Use the package manager [pip](https://pypi.org) to install ufcpy.

```bash
pip install ufcpy
```

## Usage

```python
from ufcpy import Fighter

f = Fighter('Jon Jones')

# returns 'Bones'
f.nickname

# returns 'Rochester, United States'
f.hometown
```
